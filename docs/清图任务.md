# 清图任务

## 概述

自动从左到右依次清理未通关的地图关卡。适用于推图场景，支持多种 OCR 预处理模式以应对不同的关卡卡片样式。

---

## 配置要求

> 确保分辨率大于等于 **1280×720**，否则可能无法正常使用

---

## 工作流程

1. 程序从当前关卡页面的最左侧开始识别未通关关卡。
2. 通过 OCR 识别关卡名称或通过模板匹配识别未清理标记。
3. 点击识别到的关卡进入战斗。
4. 战斗结束后自动返回关卡选择页面，继续清理下一个关卡。
5. 重复直到当前页面所有可清理关卡均已完成。

---

## 识别模式

程序支持三种 OCR 预处理模式，可根据关卡卡片的实际样式选择合适的组合：

### 普通OCR

不做任何图像预处理，直接识别关卡名称。适用场景：背景和关卡卡片颜色与白色差异明显，文字清晰可辨。

### 除杂色OCR1（标准白色）

过滤掉非纯白色区域后再识别。适用场景：关卡卡片文字为标准白色，背景较复杂干扰识别。

### 除杂色OCR2（宽松白灰色）

同时保留白色与浅灰色区域后再识别。适用场景：关卡卡片文字为白色或浅灰色，或普通OCR/除杂色OCR1均无法识别时。

---

## 选项说明

| 选项 | 默认值 | 说明 |
|------|:------:|------|
| ⭐普通OCR | True | 使用普通OCR识别关卡名称 |
| ⭐除杂色OCR1 | True | 使用标准白色过滤后识别 |
| ⭐除杂色OCR2 | True | 使用宽松白灰色过滤后识别 |

> 如果全部关闭，程序会自动回退到全部开启。
