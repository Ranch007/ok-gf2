# 周常任务

## 概述

自动完成每周例行挑战任务，从游戏主界面开始。支持各子任务独立开关。

---

## 子任务

### 首领挑战

> 检查单：「战役推进 → 模拟作战 → 首领挑战」可用。

自动进入首领挑战关卡，快速完成挑战并领取奖励。

选项：『⭐首领挑战』

### 峰值推定

> 检查单：「战役推进 → 模拟作战 → 峰值推定」可用。

自动进入峰值推定界面，先后完成：
1. **常规峰值**：领取周期报酬
2. **极限峰值**：依次挑战各等级关卡

支持自动识别并领取周期报酬奖励。

选项：『⭐峰值推定』

### 扩编实练

> 检查单：「战役推进 → 模拟作战 → 扩编实练」可用。

自动进入扩编实练界面，根据剩余体力情况判断是否挑战特殊增强关卡。

选项：『⭐扩编实练』

---

## 运行顺序

```
首领挑战 → 峰值推定 → 扩编实练
```

各子任务按上述顺序依次执行，完成后自动输出"周常完成！"通知。
