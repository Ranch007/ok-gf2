# 项目键盘操作体系说明

本项目支持键盘操作，所有按键操作均通过封装函数进行，确保代码一致性和可维护性。

---

## 1. 按键调用方式

### 1.1 方向键移动

```python
self.press_keys_sequence(keys, down_times, sleep_between=0.5)
```

- `keys`：按键列表，如 `['a', 'w', 'd']`
- `down_times`：每个按键的按住时长列表
- `sleep_between`：按键之间的间隔秒数

示例：

```python
self.press_keys_sequence(['a', 'w', 'd'], [1.087, 1.4, 0.5], sleep_between=0.5)
```

此方法通过 `keybd_event` 模拟原始按键，适用于需要精确控制按键持续时间的场景（如活动层喝水、吃饭动作）。

### 1.2 组合键点击

```python
self.click_with_key(key, box)
```

按住修饰键再点击指定位置。

示例：

```python
self.click_with_key('alt', result)
```

### 1.3 通用按键

底层通过 `send_key` / `send_key_down` / `send_key_up` 进行物理按键模拟。

## 3. 开发注意事项

- 所有按键操作应使用封装函数，避免直接调用底层 `send_key`。
- `self.press_keys_sequence` 仅接受 `w` / `a` / `s` / `d` 方向键。
- 高频重复按键场景（如活动层喝水/吃饭）已通过 `press_keys_sequence` 封装。
